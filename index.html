Instructions:

  1. Input correct answers for HL and SL in the text boxes. The format is not important as long as all characters that are not A, B, C or D will be removed before reading the answer to the system.
  2. Take pictures or scan answer sheets. Each image has to contain the paper fully with a minimum angle and difference from the template. Images of the papers of the different format are not going to be processes correctly.
  3. Convert images to PDF where each page in a file is a separate paper.
  4. Upload the PDF file to the input section and click "Grade".
  5. Near the button you will be able to see the status of grading. After it is done the output file will be automatically downloaded and named the same as input file, but with a prefix "graded_". Each page there is a graded annotated paper in a same order how it was in the input file.
